# （尚硅谷租房项目）本项目只用于本人学习使用
基于鸿蒙ArkTs以及ArkUI的租房APP

### 技术栈：
ArkTs + ArkUI + Axios

### 后端
可在Windows系统上直接运行
解压zufangBackend-windows-amd64.zip文件，点击.exe即可运行，端口号6060

### 前端配置
- 后端的.exe程序启动
- 进入entry/src/main/ets/utils/http.ets 更换baseURL为自己的ip
- 从pages中的Index.ets（程序入口）即可正常访问
